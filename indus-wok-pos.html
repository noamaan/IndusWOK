  3. Use each width below to match the correct printer/roll.

☁️ Live Multi-Device Sync

Paste your Apps Script Web App URL to share data across ALL devices (users, menu, bills, customers, etc.). Use the same URL on every device. Changes appear everywhere within ~12 seconds.

First time setup: On your MAIN device (with your menu/users) click Save then Upload My Data. On every OTHER device, paste the same URL and click Save — it pulls everything automatically. After that it stays in sync by itself.
